Getting Started with Interaction Composer

Creating a Diagram File

Now that you have created an Interaction Composer project, you can add a diagram file.

  1. Switch to the Interaction Composer perspective.
  2. Right-click on the project, and select New -> Interaction Composer ... diagram.
    3. The New Diagram creation wizard opens.
  3. Enter a name for the file and click Finish.
    4. The newly created diagram appears in the project structure. You are now ready to begin manipulating components onto the large canvas work area and linking them to one another to create an application flow.

Interaction Composer Overview

IC is a development tool and tooling environment that will allow customers to efficiently develop and deliver high quality voice agent and call center applications. IC was designed with these key factors in mind:

    Is easy to use:  This Next Generation Development Tool aids application development by providing task-oriented components for developing application solutions, such as graphical editors for developing a call-flow, or specific views and editors to facilitate creation of application elements.

    Targets developers:  This development tool supports VUI developers with familiarity of telephony applications while at the same time abstracting the technical details of implementation languages like VoiceXML. The tool also supports developers implementing server-side Java Interfaces.

    Provides extensibility and open architecture development:  While providing VUI and telephony specific enhancements, the development tool also seamlessly allows industry standard technologies and practices to be an integral part of the application delivery process or end-solution. Among the implications of this guiding principle are requirements such as Eclipse, version control, server-side programming including Java Servlet, and support for XML technologies.

    Allows software reuse:  Developers can easily incorporate existing application components including those developed through alternate environments and developers can create shareable components.
